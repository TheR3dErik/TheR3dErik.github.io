Recently, I've been obsessed with making a voxel game similar to Minecraft. One thing I've read is how commonly octrees are used for this purpose. Before I try making anything with octrees, I wanted to see if I could even use quadtrees, the more mild 2-dimensional cousins of octrees.

The canvas above is sort of like a picture in MS Paint that you can draw on, but it also visualizes the quadtree structure. In this specific case at this scale, the quadtree seems to be far less efficient than just using an array, but my sole purpose in making this was to better understand the concept. Use the mouse to draw and see how the quadtree changes.
